# 💻 Sobre o desafio

Nesse projeto, você deverá criar uma aplicação para treinar o que aprendeu com Node.js

Essa será uma aplicação onde o seu principal objetivo é uma pequena aplicação financeira. Os requisitos do sistema deverá implementar são:

- Criar uma conta
- Realizar um depósito
- Realizar um saque
- Atualizar os dados da conta
- Excluir uma conta
- Listar todas contas
- Listar as transações de uma conta
- Listar as transações de uma conta por data

## Como rodar o projeto

Após clonar o repositório e acessar a pasta do projeto, execute os comandos:

```bash
# yarn
# yarn dev
```
