# 💻 Sobre o desafio

Nesse projeto, você deverá criar uma aplicação para treinar o que aprendeu com Node.js

Essa será uma aplicação onde o seu principal objetivo é construir um backend para uma locadora de veículos. Os requisitos do sistema deverá implementar são:

- Criar uma categoria
- Listar as categorias
- Criar uma especificação
- Listar as especificações
- Documentar todas rotas da API com SwaggerUI

## Como rodar o projeto

Após clonar o repositório e acessar a pasta do projeto, execute os comandos:

```bash
# yarn
# yarn dev
```
